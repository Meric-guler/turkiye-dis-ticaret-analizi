#  Türkiye Foreign Trade Data Analysis

This project demonstrates a complete data analysis workflow using Turkey’s foreign trade statistics. It includes data cleaning, processing, analysis, and interactive visualization.

##  Project Contents
- turkiye_dis_ticaret_analizi.ipynb: Python notebook for data cleaning, calculations, and static visualizations
- turkiye_dis_ticaret.csv: Cleaned and structured raw dataset
- turkiye_dis_ticaret_raporu.pbix: Interactive dashboard built with Power BI

##  Tools & Technologies Used
- *Python*: Pandas, Matplotlib for data preparation and analysis
- *Power BI Desktop*: For interactive reporting and visualization
- *Jupyter Notebook*: For documenting the analysis process
- *GitHub*: For version control and portfolio sharing

##  Key Findings & Visualizations
- Trend of exports and imports between 2021–2025
- Top 10 countries by total trade volume
- Breakdown of trade by product groups
- Annual trade balance and deficit overview

This project is prepared to showcase practical data analysis skills, from raw data to actionable insights.
