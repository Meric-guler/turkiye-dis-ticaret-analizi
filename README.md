#  Türkiye Foreign Trade Data Analysis

This project demonstrates a complete data analysis workflow using Turkey’s foreign trade statistics. It includes data cleaning, processing, analysis, and interactive visualization.

##  Project Contents
- turkiye_dis_ticaret_analizi.ipynb: Python notebook for data cleaning, calculations, and static visualizations
- sql_sorgulari.txt: SQL queries used for filtering, grouping, and aggregating trade data
- turkiye_dis_ticaret.csv: Cleaned and structured dataset
- turkiye_dis_ticaret_raporu.pbix: Interactive dashboard built with Power BI

##  Tools & Technologies Used
- *SQL*: For data extraction, filtering, aggregation, and summary calculations
- *Python*: Pandas, Matplotlib for data preparation, validation, and analysis
- *Power BI Desktop*: For interactive reporting and visualizations
- *Jupyter Notebook*: For documenting the analysis workflow
- *GitHub*: For version control and portfolio sharing

##  Key Findings & Visualizations
- Trend of imports and exports between 2021–2025
- Trade volume breakdown by product groups
- Yearly trade balance overview
- Summary statistics using SQL and Python
